# Robotic-Sorting-System-based-on-Computer-Vision-and-TRIO

## Software
- Motion Perfect
- Python IDE
- DOBOT Robot Vision Kit

## Hardware
- 4-DOF DOBOT robot
- Hikvision industrial camera
- FLEX-6-nano and FLEX 3AX TRIO motion controllers
- Stepper Motor Driver
- I/O Modules
- Photoelectric switch
- Conveyor belts
